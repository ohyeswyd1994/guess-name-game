# 🎭 猜人名 · 局域网版

「额头贴名字」玩法的局域网网页小游戏，支持 3-6 人同时游玩。

## 🎮 玩法

1. 每个人额头上贴一个**别人给的人名**（自己看不见，但所有其他人都能看见）
2. 大家轮流提**是/否问题**，其他人投票回答
3. 任何时候觉得猜到了，点击"猜！"输入答案
4. 猜中后继续观战 + 帮其他人回答问题
5. 全部猜中后本局结束

## 🚀 启动方法

### 1. 安装 Node.js

如果还没装，去 https://nodejs.org 下载 LTS 版本安装（一路下一步即可）。

### 2. 安装依赖（首次运行）

打开命令行（Windows 按 `Win+R` 输入 `cmd`），进入项目目录：

```bash
cd %USERPROFILE%\Desktop\guess-name-game
npm install
```

### 3. 启动服务

```bash
npm start
```

启动后控制台会打印类似：

```
=== 猜人名小游戏 服务已启动 ===
本机访问:   http://localhost:3000
局域网访问: http://192.168.1.5:3000
```

把 **"局域网访问"** 后面的地址发给同事，他们用浏览器打开即可加入。

## ✅ 规则细节

- 房间唯一，第一个进入的人是**房主**，由房主点击"开始游戏"
- 至少 3 人才能开始，最多 6 人
- 出题采用**随机配对**（错位排列，每个人给一个非自己的玩家出题）
- 进入房间后可以随时**修改昵称**
- 没有提问次数限制，没有计时
- 中途有人退出会导致本局结束并回到大厅

## 🛠️ 技术栈

- 后端：Node.js + Express + ws (WebSocket)
- 前端：原生 HTML/CSS/JavaScript（无构建步骤）
- 单文件 server.js + 单文件 index.html，方便修改

## ❓ 常见问题

**Q: 同事打不开页面？**
A: 检查防火墙是否允许 3000 端口入站。Windows 第一次运行 node 时会弹窗，选择"允许专用网络"。

**Q: 想换端口？**
A: 启动时加环境变量 `PORT=8080 node server.js`（Windows: `set PORT=8080 && node server.js`）。

**Q: 想踢人/重置房间？**
A: 当前版本未实现踢人，最简单的方法是房主关掉服务再重启。
